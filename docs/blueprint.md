# **App Name**: Inmatmex Welcome

## Core Features:

- Simple Greeting: Display 'Hola Inmatmex' on the page.

## Style Guidelines:

- Primary color: Soft blue (#A0C4FF) for a calming and welcoming feel.
- Background color: Light blue (#E9EDF9), subtly desaturated for a clean look.
- Accent color: Pale violet (#D0BFFF), for highlighting the text without overwhelming the user.
- Font: 'PT Sans' (sans-serif) for a modern, readable style. Note: currently only Google Fonts are supported.
- Center the greeting text on the page for a simple, focused layout.
